# Mtan65byChatGPT

Microtan 65 developed by ChatGPT Emulador de Microtan 65 desarrollado por ChatGPT. Todo el código es generado por ChatGPT modelo GPT-5.5 de OpenAI.

Desarrollado en Java

Licencia MIT salvo excepciones en ROMs


# ROMs

Este repositorio no distribuye las ROMs originales del Microtan 65.

El usuario deberá proporcionar:

```
roms/
    TANBUG.BIN
    CHARSET.BIN
```

Los derechos de dichas ROMs pertenecen a sus respectivos titulares.

Este proyecto únicamente proporciona el emulador.
